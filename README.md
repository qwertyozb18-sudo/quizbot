# QuizBot

Telegram Quiz Bot with Web App integration.

## Features
- Quiz matching (English, Russian, Math, Physics)
- Web App Dashboard (Statistics, Ranking, Exchange)
- PostgreSQL Database
- Admin Panel

## Tech Stack
- Python (Aiogram 3, Aiohttp)
- PostgreSQL (Asyncpg)
- HTML/JS (Frontend)

## Deployment
Deployed on Railway.
